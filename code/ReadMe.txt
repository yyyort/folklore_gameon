Controls

W = UP
A = LEFT
S = DOWN
D = RIGHT

Q = Weapon Swap (4 Items)
E = Support Swap (2 Items)
R = Equipment Swap (1 Item)

SPACE = Weapon Attack (Melee)
LCTRL = Use Support (Fire Magic is not implemented DO NOT use)
                    (Heal Magic is USABLE)

F = User Equipment (Molotov, concept USABLE)
